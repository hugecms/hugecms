# HugeCMS 前台主题上下文

HugeCMS 前台主题系统负责将内容以不同的视觉与布局形式呈现给访客。管理员可在后台为整站启用一个主题，主题由 Blade 模板、样式、脚本和元数据组成。

## Language

**主题（Theme）**：
一组命名且自包含的前台呈现资源，包括 Blade 模板、CSS、JS 以及描述自身的元数据。一个主题对应一个独立的目录。
_Avoid_: 模板包、皮肤、外观

**当前主题（Active Theme）**：
管理员为整站启用的主题，所有未指定其他主题的访客请求均使用该主题渲染。
_Avoid_: 使用中的皮肤、生效模板

**默认主题（Default Theme）**：
当当前主题缺少某个视图时，系统回退使用的主题。默认主题必须完整实现前台所需的所有视图。
_Avoid_: 基础模板、兜底皮肤

**主题目录（Theme Directory）**：
存放单个主题全部内容的统一目录，包含 `views/` 子目录、`css/` 与 `js/` 入口以及 `theme.json`。例如 `resources/themes/default/`。
_Avoid_: 模板目录、资源目录

**主题预览（Theme Preview）**：
管理员通过在 URL 中附加 `?theme={theme}` 参数临时查看尚未启用的主题效果；普通访客无法使用该能力。
_Avoid_: 预览模式、临时皮肤

**主题管理页（Theme Management Page）**：
后台中独立于站点设置的页面，用于展示已发现的主题卡片、查看编译状态、切换当前主题。
_Avoid_: 主题设置、皮肤管理

**主题包元数据（Theme Metadata）**：
描述主题名称、展示标题、预览图、作者、版本等信息的 `theme.json` 文件，供后台主题列表展示使用。
_Avoid_: 主题配置、皮肤信息
